# Juego del Ahorcado

Juego del ahorcado desarrollado con **React** y **TypeScript** como práctica de desarrollo frontend.

## Descripción

El usuario debe adivinar una palabra oculta ingresando letras. Cada error reduce el número de intentos disponibles hasta ganar o perder la partida. Utiliza imágenes que van dibujando al muñeco.

## Tecnologías

- React  
- TypeScript  
- CSS
